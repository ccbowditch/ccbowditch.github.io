Learn the Fundamentals

At Coding Club, we teach the fundamentals of web development. We teach languages including but not limited to: Javascript, HTML, CSS.

HTML, CSS, and Javascript construct most of the code on the front end side of web development. This is the 'showcasing' code. It is what is seen by end clients. We also teach some backend code. Mainly NodeJS which is a Javascript runtime. This is known as the 'behind the scenes' code. We will focus mainly on Javascript as it is the most popular web development language.

Learn Frameworks and Libraries

At Coding Club, we also teach libraries and frameworks of mainly Javascript.

Frameworks and Libraries are 'extensions' of already made languages. E.G Javascript. Frameworks and Libraries are important in the use of web development since it helps to shorten the time you spend to make an application or helps you make your application better in different ways. Some that we will be teaching is Webpack(Front end), Websockets(A mix of front end and backend), and Express(Back end).

About Coding Club

Why it was formed?

Coding Club was formed to give people a more immersed programming experience. Other than scratch people will be able to code using modern languages. It would be fun to watch people learn programming and be able to develop complex applications.
